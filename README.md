# visão do projeto leitor de feeds.
Nesse projeto que é em leitor de teste de feed que foi baseado na web que lé feeds RSS, que utiliza a ferramenta jasmine para escrever os testes da aplicação.

# Os Testes Feitos na aplicação.
  ## Teste suíte “RSS feed”
     * faz um loop em cada feed no objeto allFeeds e garanta que ele tenha uma URL definida e não esteja vazia.
     * faz um loop em cada feed no objeto allFeeds, e garanta que ele tenha um nome definido e não esteja vazio.
  ## Teste suíte "O menu".
    * garanta que o elemento de menu permaneça oculto por padrão
    * garanta que o menu altere a visibilidade quando alguém clica no menu.  menu deve ser exibido quando alguém clicado uma vez nele e ocultado quando clicado duas vezes.
  ## Testes suíte "Entradas iniciais".
    * garanta que, quando a função loadFeed for chamada e concluir seu trabalho, exista pelo menos um elemento .entry no contêiner .feed.
  ## Teste suíte "Nova seleção de feed".
    * garanta que, quando um novo feed for carregado pela função loadFeed, o conteúdo realmente mude.

# Como visualizar o projeto Teste de Feed. 
Para visualizar o projeto basta abrir o arquivo index.html localizado na pasta raiz como o navegador de sua preferência.
